# Austim-detection
The aim of this research is to employ advanced topic modeling techniques to uncover hidden topics within Machine learning and Deep learning .The purpose of the research is to examine the possibility of using artificial neural networks (ANNs), a type of deep learning, to diagnose autism spectrum disorder (ASD). The goal is to create an ASD detection technique that is quicker and easier to use than current approaches. In order to assess survey data and determine a person's probability of having ASD, the study suggests using an ANN model. Metrics such as the confusion matrix, accuracy, recall, precision, and F1 score are used by the authors to assess the model's accuracy. The efficacy of the strategy is then confirmed by contrasting the outcomes with real-time data.
